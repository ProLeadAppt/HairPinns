# Retail SEO Checklist

Keep Hair Pinns visible for national retail queries: "hair products Australia", "salon hair products Australia", "hair care Australia", "best hair products Australia". This checklist is retail-specific; for local salon SEO see [GMB Sync Checklist](gmb-sync-checklist.md).

---

## National Retail Keyword Targets

| Keyword | Target Pages | Notes |
|---------|--------------|-------|
| hair products Australia | Homepage, Collections, Product pages | In title, meta, schema |
| hair products Australia only | Homepage FAQ, Shipping page | Australia-only messaging |
| buy hair products Australia | Homepage, Collections, Search | Content, meta |
| salon hair products Australia | Collection pages, Product pages | OG titles, meta descriptions |
| hair care Australia | Homepage, Collections index, Blog | Content, FAQ schema |
| best hair products Australia | Blog ("Best Hair Products Australia 2025"), Homepage FAQ | ✅ Done |
| free shipping Australia | Shipping policy, Homepage | Policy page SEO |
| hair products Melbourne / Brisbane / Perth | Blog ("Hair Products Melbourne Brisbane Perth"), Shipping page | ✅ Done |

---

## Quarterly Tasks

1. **GSC Export** — Performance data for product/collection queries
   - Filter by: pages containing `/products/`, `/collections/`, `/search`
   - Export queries + pages as CSV
2. **CTR Analysis** — Use [GSC CTR Analysis Workflow](gsc-ctr-analysis-workflow.md) for title tag opportunities
3. **Competitor Watch** — Oz Hair & Beauty, Beautopia, My Haircare
   - What do they rank for? (manual search or tool)
   - Any schema or content gaps we can address?

## Schema Checklist

- [x] Store schema on homepage and Collections index (`generateStoreSchema`)
- [x] Product schema with `shippingDetails` (Australia-wide)
- [x] CollectionPage schema on collection detail pages
- [x] ItemList schema on Search Results when products exist
- [x] ItemList schema on Blog index (article listings)
- [x] FAQ schema with "hair products Australia" question on homepage

---

## Title & Meta Checklist

- [x] Homepage: "Australia" in title
- [x] Collections index: "Hair Pinns Australia" / "Hair Products Australia"
- [x] Collection detail: "Hair Care Australia" in title
- [x] Product pages: "Hair Care Australia" in title
- [x] Search results: "Hair Products Australia" when query present
- [x] Shipping policy: "Free Shipping Australia-Wide"
- [x] Returns policy: "Hair Product Returns Australia"

## Australia-Only and State Coverage

- [x] Australia-only shipping messaging on Shipping page and FAQ ("Do you ship internationally?")
- [x] State/territory coverage in Shipping page for long-tail (Melbourne, Brisbane, Perth, Gold Coast, Newcastle, Wollongong)
- [x] Store schema description includes "Australia-only shipping"

## Internal Linking

- [x] Footer: "Shop Hair Products Australia-Wide" → /collections
- [x] Suburb pages: CTA to /collections for retail
- [x] Collection pages: Cross-links to related collections
- [x] Blog product modules: Links to /collections with Australia anchor text where natural
